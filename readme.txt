== Description ==

Used to overlay videos from Youtube, Metacafe, Google Video, iFilm and custom flv files on the current page.

== Installation ==

1. Upload the folder "videobox" to the "wp-content/plugins" directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. It's done ! Now, you need to insert a link like below :
href="http://www.youtube.com/watch?v=VhtIydTmOVU" rel="vidbox" title="caption"

Upgrade

Uninstall the previous version and follow the Installation instructions.

== Frequently Asked Questions ==

= Can i install the plugin whit Lightbox plugin ? =

Actually it's not possible to run Videobox when Lightbox plugin is activate.
